Use this as the working target sheet for roles in agentic AI, AI infrastructure, inference systems, developer tools, robotics AI, data infrastructure, evals, observability, model serving, and ML platforms.

## Sponsorship data sources

| Source | Use it for | Link |
| --- | --- | --- |
| USCIS H-1B Employer Data Hub | Official employer petition history. Best first check for whether an employer has filed H-1B petitions. | https://www.uscis.gov/tools/reports-and-studies/h-1b-employer-data-hub |
| MyVisaJobs employer search | Search employer LCA, H-1B, PERM, salaries, and historical role titles. | https://www.myvisajobs.com/employers/search.aspx |
| Levels.fyi H-1B salary database | DOL H-1B salary records by company, title, location, and year. | https://www.levels.fyi/h1b/ |

Search strings:

```text
site:company.com/careers "visa sponsorship"
"COMPANY" "H-1B" "Software Engineer" "MyVisaJobs"
"COMPANY" "H-1B" "Machine Learning Engineer"
"COMPANY" "OPT" "STEM OPT" "new grad"
"COMPANY" "Software Engineer" "new grad" "sponsor"
```

## Daily Hermes brief format

Hermes should return a concise morning brief with:

1. New or changed roles in the target lanes, grouped by tier.
2. Direct apply links and the company's official careers link.
3. Sponsorship signal: confirmed H-1B history, unclear, or likely risky.
4. Role fit tags: AI infra, inference, agents, robotics, evals, data infra, devtools, new grad.
5. Next action: apply, find referral, ask recruiter about sponsorship, or skip.
6. One short outreach line when a role is high-fit.

Do not include crypto market updates.
